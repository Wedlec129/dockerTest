Start/pause/unpause

Контейнер типо сборка из образов(питона си и тд типо набор программ в одной коробке)

docker pull openjdk 	скачать образ openJDK(java)

Если мы попытаемся запустить образ то автоматически создаст докер контейнер


docker run openjdk 		запустит образ (и создаст контейнер) , но скорее всего и сразу выйдет 
docker run -it openjdk 	запустит образ интерактивно то есть будет раб пока мы не выйдем
docker run -it --name myJava openjdk  запустит образ + добав имя								!!!!!!
docker start/stop/pause/unpause/kill myJava 	запуск контейнера



Можно делать конфигурационные файлы 


docker build .			собрать конф файлы
